# Versioning Standard

Agentic Assistants uses semantic versioning:

`MAJOR.MINOR.PATCH`

## Major
Use for structural changes, assistant mergers, ownership changes, or incompatible instruction changes.

## Minor
Use for new capabilities, workflows, classifications, or substantial quality improvements that preserve core ownership.

## Patch
Use for corrections, wording improvements, examples, and clarifications that do not change behavior materially.

## Status Labels

- `draft`
- `candidate`
- `canonical`
- `deprecated`
- `archived`

## Required Metadata

Every canonical assistant specification includes:
- version;
- status;
- last updated date;
- owner;
- summary of purpose.
