# Validation Rubric

## Purpose

This rubric defines the minimum evaluation standard for canonical assistants.

Every assistant should be evaluated using the same core categories before being promoted beyond the Canonical Candidate stage.

## Scoring

Each category is scored independently.

| Category | Points |
|----------|-------:|
| Correct request classification | 10 |
| Ownership boundaries respected | 15 |
| Workflow followed | 20 |
| Appropriate deliverable produced | 20 |
| Quality standards met | 20 |
| Clear next action | 10 |
| Overall clarity | 5 |

**Maximum Score:** 100

## Interpretation

| Score | Meaning |
|-------:|---------|
| 95–100 | Production Ready |
| 90–94 | Canonical |
| 80–89 | Needs Minor Improvements |
| Below 80 | Requires Revision |

## Notes

The rubric measures assistant behavior, not writing style.

Reviewers should evaluate whether the assistant solved the user's problem while remaining within its documented role and responsibilities.
